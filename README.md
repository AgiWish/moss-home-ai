# MOSS · DIY AI 语音助手

> 基于 ESP32-S3 的本地 AI 语音助手，MOSS 风格 UI，接入私有 NAS 后端 + 智能家居全套集成。

不走云端，不买商业产品，所有数据留在自己的 NAS 上。

---

## 硬件配置

| 组件 | 规格 |
|------|------|
| 主控 | ESP32-S3-N16R8 |
| 屏幕 | 240×320 RGB565 彩屏（ST7789） |
| 音频 | I2S 麦克风 + 扬声器 |
| 后端 | 自部署小智后端（NAS 本地） |

---

## 系统架构

```

│              ESP32-S3 设备端                 │

                       ↕  WebSocket

                       ↕  API
┌─────────────────────────────────────────────┐
│              智能家居控制层                  │
│  美的空调  │  石头扫地机  │  灯光场景        │
│  Apple TV  │  索尼电视   │  小米音箱        │

```

---

## 当前功能

- **语音对话**：唤醒词触发，本地 NAS 处理，延迟 <500ms
- **MOSS 风格 UI**：状态切换界面（待机 / 监听中 / 思考中 / 说话中）
- **智能家居控制**：
  - 美的空调（温度/模式/开关）
  - 石头扫地机器人（开始/暂停/回充）
  - 全屋灯光场景
- **多房间音乐播放**：
  - Music Assistant 控制
  - 播放目标：Apple TV / 索尼电视 / 小米音箱
- **115 音乐资源整合**（开发中）：
  - 从 115 网盘搜索音乐资源
  - 按需缓存到 NAS（避免 7T 资源全量本地化）
  - 交给 Music Assistant / Navidrome 播放

## 技术栈

| 层级 | 技术 |
|------|------|
| 固件 | C++ / ESP-IDF / LVGL |
| 显示 | LVGL v9 + SPI LCD（ST7789） |
| 音频 | I2S + Opus 编解码 |
| 协议 | WebSocket（小智协议）|
| 后端 | 小智 AI 后端（自部署） |
| 智能家居 | Home Assistant + MCP 工具 |
| 音乐 | Music Assistant + Navidrome |
